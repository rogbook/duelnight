# 게이머 UI 2차 — 확정 시안 명세서 (사용자 승인 2026-06-12)

> **이 문서가 2차 UI 작업의 단일 기준이다.** 1차(부분 손질)와 달리 홈 화면과 내비게이션을 **전면 교체**한다.
> 대원칙: [DESIGN_DIRECTION.md](./DESIGN_DIRECTION.md) §디자인 대원칙. 담당: Antigravity. 검토: Claude.

## 0. 목표 한 줄

SaaS 대시보드(페이지 제목 + 흰 박스 + 텍스트 사이드바) → **게임 동반 앱**(내 프로필이 주인공 + 살아있는 숫자 + 컬러 타일 + 하단탭/아이콘레일).

## 2. 공통 컴포넌트 (5개 신규 — `src/components/game/`)

1. **ProfileBar** — 아바타(38px 원형, `--game-blue-deep`) + 닉네임 + 티어 배지(금색 칩, `user_ratings` 기반·없으면 숨김) + XP/진행바(5px, `--game-blue`) + 알림 벨. 모바일·태블릿·PC 공용.
2. **SeasonStatWidget** — "이번 시즌 승률" 라벨 + 대형 숫자(24~30px) + 연승 표시(`--game-win`) + 전/승/패 칩 3개(`--game-bg` 배경 알약). 데이터는 기존 SeasonReport 쿼리 재사용. **전적 0건이면**: 숫자 대신 "첫 대전을 기록해 보세요" + CTA 버튼(전적 기록하러 가기).
3. **MenuTile** — 아이콘 칩(36~40px, 위 4색 순환) + 이름(12px, 500) + **살아있는 숫자 한 줄**(카드 3,841장 / 덱 N개 / 오늘 N판 / 모집중 N명 — 기존 쿼리 재사용, 없으면 설명 문구). lucide 아이콘만 사용.
4. **BottomTab** (모바일+태블릿) — 5개: 홈 / 카드 / **대전(중앙, 44px 원형 떠있는 버튼, `--game-blue-deep`, 위로 -22px)** / 덱 / 프로필. 활성=`--game-blue`, 비활성=`--game-icon-idle`. 기존 bottom-tab-bar.tsx 개조.
5. **IconRail** (PC 전용, 기존 사이드바 대체) — 54px 폭, 로고 마크 상단, 메뉴 아이콘 세로 배열. **호버 시 이름+한줄설명 툴팁**(기존 shadcn tooltip 재사용). **하단 펼침(») 토글** → 160px 폭, 아이콘+이름 표시. 펼침 상태 localStorage 기억, **첫 방문 기본값 = 펼침**. 관리자 메뉴는 기존처럼 isAdmin일 때만.

## 3. 기기별 홈 레이아웃

- **모바일(<768px)**: ProfileBar → SeasonStatWidget(전폭) → MenuTile 2×2 → BottomTab. 사이드바 없음.
- **태블릿(768~1024px)**: ProfileBar → 스탯 3칸(승률 1.4fr + 리그 + 오늘) → MenuTile 4열 → BottomTab 유지.
- **PC(>1024px)**: IconRail + 본문[ProfileBar(우측에 "시즌 종료까지 N일") → 승률(1.5fr)+리그(1fr) → MenuTile 4열 → 인기 덱 한 줄(`이번 주 인기 덱 — OO 승률 N% · 사용자 N명`, 티어리스트 데이터, 없으면 숨김)].

## 4. 불변 규칙 (위반 시 반려)

- **로직·라우트·데이터 쿼리 변경 금지** — 표현(컴포넌트·스타일)만. 기존 기능(알림·언어선택·관리자 배지 등) 전부 보존.
- **관리자 콘솔(/admin)·카드 등록 화면 변경 금지** (SaaS 유지 확정).
- 새 패키지 금지, lucide 아이콘만. 애니메이션은 transform/opacity, blur는 하단탭 1곳, prefers-reduced-motion 존중.
- 빈 상태(전적 0, 덱 0, 티어 없음) 모두 자연스럽게 처리 — 깨진 숫자·NaN 노출 금지.
- i18n 키 유지(신규 문구는 ko/en/ja 3개 다 추가).
- 한국어로 계획·보고, **브랜치에 push 후 보고** (main 직접 push 금지).

## 5. 작업 순서

1. **1단계(이번 차수)**: 디자인 토큰 + 공통 컴포넌트 5개 + 홈 3레이아웃 + IconRail/BottomTab 전환. 완료 보고 → Claude 검토·병합 → 사용자 폰 평가.
2. **2단계(다음 차수)**: 카드 도감·덱·전적 화면을 같은 톤으로. (별도 지시)
