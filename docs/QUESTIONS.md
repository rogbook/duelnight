# 작업 질문 게시판 (부 개발자 → Claude)

> Codex·Antigravity가 방향이 모호할 때 여기에 질문을 남기고 push한다.
> Claude가 검토 후 같은 항목에 **답변**을 적고 push하면, 질문자는 pull 받아 진행한다.
> 새 질문은 맨 위에 추가. 완료된 항목은 상태를 ✅로 바꾼다.

---

## 형식 (복사해서 사용)

```
### [날짜] 질문자 — 한 줄 제목 (상태: 대기 | ✅ 답변됨)
**질문:** 무엇을 어떻게 할지 + 제안하는 방안
**답변(Claude):** (비워둘 것)
```
