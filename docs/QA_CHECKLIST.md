# tcg-hub QA 체크리스트 (Publish 전 필수)

> Preview URL에서 관리자 계정으로 아래 항목을 모두 통과해야 Publish 가능.
> 통과 시 항목 옆에 ☑, 실패 시 이슈 번호/원인 메모.

## 1. 인증
- [ ] 비로그인 상태에서 `/intro` 정상 노출
- [ ] 이메일/비밀번호 로그인 성공
- [ ] Google 로그인 성공
- [ ] 비밀번호 재설정 이메일 수신
- [ ] 로그아웃 후 보호 라우트 진입 시 `/login` 리다이렉트

## 2. 카드 DB
- [ ] 카드 목록 페이지네이션/검색/필터 동작
- [ ] 카드 상세 페이지 이미지·텍스트 정상 표시
- [ ] (관리자) 카드 등록: 이미지 1장
- [ ] (관리자) 카드 등록: 이미지 2장 이상 (얼터 포함)
- [ ] (관리자) 카드 수정: 이미지 추가/삭제
- [ ] (관리자) 카드 수정: 코드 변경 시 일러스트도 함께 이동
- [ ] (관리자) 카드 일괄 업로드 CSV/JSON

## 3. 덱
- [ ] 덱 빌더 카드 검색·추가
- [ ] 덱 저장·수정·삭제
- [ ] 덱 레시피 편집 (이미지 누락 카드 "DB 미등록" 배지 노출)
- [ ] 공개 덱 공유 링크 접근

## 4. 매칭
- [ ] 전적 기록 추가/수정/삭제
- [ ] 승률 차트 정상 렌더
- [ ] 리더보드 정상 로딩

## 5. 오프라인 매칭 (LFG)
- [ ] 게임/카테고리/상태 필터 클릭 응답
- [ ] 게시글 작성·수정·삭제
- [ ] 카드 클릭으로 상세 진입
- [ ] 댓글 작성·삭제, 신고

## 6. 매장·이벤트·캘린더
- [ ] 매장 목록·상세
- [ ] 이벤트 즐겨찾기
- [ ] 캘린더 ICS 내보내기

## 7. 결제 (sandbox)
- [ ] 크레딧 충전 결제 sandbox 성공
- [ ] 결제 실패 케이스 토스트
- [ ] 결제 내역 조회

## 8. 알림·공지
- [ ] 알림 벨 카운트 갱신
- [ ] 공지사항 목록·상세

## 9. 권한
- [ ] 비관리자 계정이 `/admin/*` 진입 차단되는지
- [ ] 관리자 콘솔 사용자 권한 부여/해제
