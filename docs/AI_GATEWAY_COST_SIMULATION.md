## 6. 권고 (PRICING_REVIEW.md 업데이트)

마크업이 **0** 으로 확정되었으므로 PRICING_REVIEW의 결론을 다음과 같이 강화/수정:

1. **가격 인상 불필요** — 확정 (구글 정가 = Lovable 청구가).
2. **무료 OCR 5회/일 유지 가능** — 한도 내 최대 사용해도 인당 월 ₩245, 유입 비용으로 합리적.
   - 단, 회원당 일 사용량을 1,000회 이상 칠 가능성이 있다면 일 한도를 **10회/일** 정도로 명시적 캡으로만 보강.
3. **"Pro 무제한" → "공정사용 상한"** — 권장 강도는 "필수"에서 **"권장(우선순위 중)"** 으로 완화 가능.
   - 일반/헤비 사용자는 마진 88% 유지. 일 100회+ 악용자만 차단하면 충분.
   - 예: Pro OCR 일 30회 / 월 500회 소프트캡 + 초과시 안내.
4. **코치 단가 인하 여지** — 코치 10C(₩85)는 원가 ₩1.8 대비 47배. **5C(₩40)** 정도로 인하해 사용 촉진 + Pro 차별화에 활용.
5. **카드 가져오기 (관리자)** 는 호출당 ₩22로 가장 비싸므로 **관리자 전용 유지 필수**. 일반 노출 절대 금지.
